PROJECT 07 — EXPERIENCE DIRECTION BIBLE
THE AVANT-GARDE TRANSFORMER
0. WHAT THIS DOCUMENT IS

This is not a generic sitemap brief and not a normal salon website planning document.

This is the production experience bible for building The Avant-Garde Transformer as a true cinematic portfolio world.

Its purpose is to control:

the emotional architecture of the site
the visual and spatial behavior of every major section
the motion language
the image sequencing system
the interaction signatures
the anti-template rules
the “wow factor” set pieces
the hierarchy of what must feel expensive, rare, and unforgettable

This world should not look like a polished HTML/CSS portfolio.

It should feel like:

a fashion transformation film
a conceptual beauty installation
an image laboratory
a sculptural editorial system
a high-fashion metamorphosis experience
1. WORLD POSITIONING
Core idea

The Avant-Garde Transformer is about hair as metamorphosis, distortion, silhouette engineering, and image reinvention.

Not wearable beauty.
Not glam.
Not haircut discipline.
Not salon luxury.
Not “creative styling” in the weak Pinterest sense.

This world is for the version of hair practice that:

mutates identity
alters silhouette
changes the emotional reading of a face
turns beauty into concept
uses hair as sculpture, costume, architecture, abstraction, and performance surface
treats the human head as a site for transformation, not decoration

The site must feel like a place where hair is no longer simply styled.

It is:

constructed
engineered
manipulated
disguised
amplified
weaponized
abstracted
reborn
2. THE EXPERIENCE MUST FEEL LIKE THIS

The site should feel like entering a private transformation chamber inside a luxury fashion universe.

Not a “homepage.”

A controlled visual event.

The user should feel:

First 5 seconds
intrigue
tension
disorientation
visual seduction
uncertainty about where the image ends and the silhouette begins
First 30 seconds
this stylist doesn’t “do hair”; they change identities
this project belongs in editorials, fashion film, runway, campaign work, art-direction collaborations
the work is technically serious, not random weirdness
the atmosphere is expensive, not costume-y
By the midpoint of the experience
the visitor understands the site has its own visual physics
every section feels like a new phase of transformation
the work is theatrical, but disciplined
the image system is curated like a gallery, not dumped like a portfolio
By the end
the visitor feels they’ve passed through a cinematic fashion ritual
the stylist appears not as a service provider but as a transformation director
the project feels singular, memorable, and impossible to confuse with another portfolio
3. NON-NEGOTIABLE EXPERIENCE SIGNATURE

If this project is successful, it should be remembered for three things:

Signature 01 — Silhouette before face

The visitor should repeatedly encounter imagery where the shape, contour, mass, obstruction, distortion, or outline of the hair form hits before the face does.

The silhouette should often be the primary dramatic event.

Signature 02 — Transformation as sequence, not gallery

The site should not feel like “here are images of avant-garde hair.”

It should feel like the user is moving through:

state 1
state 2
distortion
reconstruction
reveal
mutation
aftermath

The experience itself must feel like metamorphosis.

Signature 03 — Editorial severity with cinematic luxury

The world must be experimental, but never messy.

It should feel:

exact
expensive
art-directed
emotionally cold or intense in the right places
high-fashion rather than alternative subculture

If it drifts toward “edgy creative hair portfolio,” it has failed.

4. WHAT THIS SITE IS ABSOLUTELY NOT

The build must not feel like any of the following:

an edgy salon site with black backgrounds
a generic creative-portfolio template with fashion photos inserted
a costume or cosplay transformation page
a beauty site with one “weird” editorial project layered on top
a chaotic art-school experiment with no hierarchy
a trend-chasing brutalist portfolio
a gothic gimmick site
a “creative director moodboard” without actual experience design discipline
a Webflow luxury template with oversized serif text and parallax pasted on

This world must be more rigorous than “creative.”
It must be designed like a controlled image machine.

6. THE SITE SHOULD BE BUILT LIKE A FILM IN 8 ACTS

Do not think of this as “10 website sections.”

Think of it as an 8-act transformation film.

ACT 01 — THRESHOLD / ENTRY INTO THE LAB
Purpose

Break the visitor’s normal expectations immediately.

This act should create:

visual tension
identity ambiguity
curiosity
a feeling of entering a private conceptual space
What happens here
hero image or cinematic canvas dominates
typography is minimal and severe
the site introduces the project as a world, not a service page
the first motion beat establishes the project’s physics
Emotional note

The user should feel like they’ve crossed a threshold into a controlled experimental chamber.

ACT 02 — THE MANIFESTO OF TRANSFORMATION
Purpose

State the thesis:
hair can alter silhouette, identity, and narrative.

What happens here
short manifesto blocks
layered editorial text composition
one or two transformation portraits or detail crops
copy behaves like a gallery wall text merged with a fashion campaign statement
Emotional note

This is where the visitor understands this project has an intellectual core.

ACT 03 — THE SILHOUETTE CHAMBER
Purpose

Show that the work is about shape, not just styling.

What happens here
large full-screen or near full-screen silhouette studies
side profiles, back views, obstructed faces
scrolling should feel like moving through sculptural installations
Emotional note

This act should feel like walking around hair sculptures in a dark gallery.

ACT 04 — THE MUTATION ARCHIVE
Purpose

Present the strongest transformation work in a curated, escalating sequence.

What happens here
editorial gallery system
alternating full-bleed images, stacked diptychs, vertical studies, detail interrupts
the image rhythm should feel authored, not uniform
Emotional note

This is the “proof of power” act.

ACT 05 — THE CONSTRUCTION LAB
Purpose

Reveal technical credibility.

This is where the user sees that the work is not accidental, not costume, and not merely aesthetic posturing.

What happens here
construction process
wig / extension / structure / pinning / sculpting evidence
macro detail shots
annotated conceptual phases if useful
Emotional note

The visitor should feel respect for the technical craft.

ACT 06 — THE IDENTITY SPLIT
Purpose

Show transformation as state change.

This is the before/after logic of the site—but done in an editorial, cinematic way rather than a salon proof widget.

What happens here
split states
sliding or layered reveals
pre-transformation vs transformed form
visual identity rupture
Emotional note

This act should feel eerie and powerful, not commercial.

ACT 07 — THE ATMOSPHERE INTERLUDE
Purpose

Deepen the world with non-human imagery and environmental clues.

What happens here
still lifes
wigs on stands
clips, combs, lacquered surfaces
shadow, mirrors, industrial textures
fragments of garments or backstage materials
Emotional note

This act is a breath—but still loaded with tension.

ACT 08 — THE INVITATION TO TRANSFORM
Purpose

Close the world without collapsing into “book now” salon language.

What happens here
collaboration CTA
transformation consultation CTA
final statement
the site should close like the final frame of a fashion film, not like a footer
Emotional note

The final feeling should be:
If I want reinvention, I go here.

7. HOMEPAGE EXPERIENCE ARCHITECTURE

Below is the actual homepage experience structure to build toward.

SECTION 01 — HERO / THE TRANSFORMATION GATE
Objective

Create a first screen that immediately announces:

conceptual authority
transformation
silhouette obsession
luxury image direction
Required build behavior

The hero must not be a standard full-screen image with centered heading.

It should be one of these high-end systems:

Option A — Layered cinematic hero
full-viewport portrait or silhouette image
foreground typography locked in asymmetrical position
secondary detail crop floating or pinned in a different depth plane
subtle image drift / parallax / mask motion
hero feels like an editorial cover with internal depth
Option B — Transformation diptych hero
left side: obscured or “pre-state” identity
right side: transformed silhouette
scroll causes subtle relationship shift between the two
could be horizontal split, staggered overlay, or one image masked through another
Option C — Black-box hero with reveal
opening frame is mostly dark, with only fragments visible
silhouette or hair form gradually reveals through motion / scroll / mask / cursor proximity
extremely effective if image quality is elite
Hero content rules

The hero must foreground one of the following:

sculptural silhouette
face obscured by hair-object form
dramatic side profile
hair construction that reads as architecture
a look so conceptually strong it can carry the first 8–12 seconds alone
Hero copy rules

Copy should be minimal.
Do not explain everything.

Use:

project title
one transformation statement
perhaps one supporting line

No generic intro paragraph.

SECTION 02 — THE MANIFESTO STRIP
Objective

Introduce the conceptual thesis without killing momentum.

Spatial design

This should not be a standard text block.

Build it as:

staggered editorial text
split across columns
perhaps one phrase pinned while the rest scrolls
text fragments can feel like wall labels in a gallery
Visual pairing

Pair with:

one brutal close crop
one detail of hair construction
one side-profile silhouette
Motion idea

As the user scrolls, different fragments of the manifesto come into dominance while imagery crossfades or slides in layered depth.

SECTION 03 — SILHOUETTE GALLERY / THE CHAMBER OF FORMS
Objective

Establish that the work is about shape architecture.

This section should feel like

A corridor of transformation statues.

Layout direction

Use one of these structures:

Structure A — Vertical procession

A sequence of tall, almost-monolithic portraits with massive negative space around them.

Structure B — Offset sculptural grid

A broken grid where some portraits are oversized, some are narrow side crops, and the eye has to navigate shape rather than simple cards.

Structure C — Sticky silhouette stage

A pinned viewport where one silhouette remains dominant while secondary transformation studies slide past it.

Image rules

Use:

side profile
back view
shape against neutral background
face partially hidden
hair form reading as mass and line

Do not clutter with too many images at once.

SECTION 04 — MUTATION ARCHIVE / MAIN PORTFOLIO BODY
Objective

Deliver the richest editorial sequence of transformed looks.

This is the core portfolio body of the site.

Critical rule

This must not become a normal masonry gallery.

It needs an authored image rhythm.

Gallery sequencing system

The image rhythm should alternate between:

one dominant full-width transformation frame
two supporting close crops or detail studies
one vertical portrait
one empty / text / quote / number interrupt
one pair or triptych of related mutation studies
one process or silhouette interruption
repeat with variation

The goal is to create the feeling of a fashion book or museum catalog, not a gallery plugin.

Interaction rule

Images should feel “held” in place by the page architecture.
Avoid noisy hover overlays.

Hover can reveal:

transformation title
concept keyword
shoot type
year / collection / editorial note

But it must remain minimal.

SECTION 05 — CONSTRUCTION LAB / HOW THE TRANSFORMATION IS BUILT
Objective

Turn the visitor’s awe into trust.

This section proves:

technical rigor
structural experimentation
serious craft
repeatable methodology
It should include
structure planning imagery
hands pinning / shaping / building
wig prep or hairpiece construction
sectioning or form manipulation
lacquer / finish / edge detail
support systems, materials, and shape engineering
Design direction

This section should feel more forensic than the rest of the site.

Possible visual treatment:

darker backgrounds
more modular composition
smaller detail frames alongside one dominant process image
subtle captioning or micro-annotation
a feeling of the viewer being allowed into the lab
Important

Do not make it feel like a tutorial or “how I work” salon explainer.

It should feel like viewing a transformation dossier.

SECTION 06 — THE IDENTITY SPLIT / TRANSFORMATION PROOF
Objective

Show the actual rupture between states.

This section should not use a generic salon before/after slider.

That would destroy the project.

Instead use one of these systems:

Option A — Layered dual-state frame

The same subject in two states, stacked or overlaid, with scroll-driven reveal.

Option B — Split-screen identity table

One side = pre-transformation studio state
Other side = final transformed identity
Typography in between acts like evidence labels.

Option C — Drag reveal with cinematic restraint

A custom drag interaction, but built like an art exhibit—not a “wow makeover” tool.

Tone rule

This section must feel psychologically charged, not commercially demonstrative.

Use captions like:

silhouette shift
facial framing altered
line extended
texture re-authored
profile architecture rebuilt

Not:

“before and after”
“what a difference”
“from plain to glam”
SECTION 07 — ATMOSPHERE / STILL-LIFE INTERLUDE
Objective

Make the world feel bigger than the portraits.

This section should include
wigs on stands
combs, clips, pins
mirrored surfaces
mesh, plastic, vinyl, gauze, shadow
backstage tables
mannequin fragments if done elegantly
detail shots of tools and material surfaces
Why this matters

Without this, the project risks feeling like a portfolio of isolated photos.
This section gives it world-building density.

Layout direction

This can be:

horizontal strip
staggered still-life wall
image-and-quote interlude
pinned atmospheric sequence
SECTION 08 — THE COLLABORATION INVOCATION
Objective

Convert without collapsing the world.

This is not a booking box.
It is the final invitation into transformation.

CTA language should feel like:
begin a transformation project
commission an editorial reinvention
build a conceptual hair world
book a transformation consultation
CTA layout ideas
Option A — Closing cinematic portrait + minimal form
Option B — Black space + one severe line + elegant CTA block
Option C — Contact panel beside a final transformation image fragment
Never do this
bright CTA buttons everywhere
standard “book an appointment” salon footer
testimonials and map widget jammed under the closing section
social media clutter
8. THE WOW FACTOR MOMENTS THIS SITE MUST CONTAIN

The project needs set pieces—moments that people remember.

At least 4 of the following 7 should make it into the final build.

WOW MOMENT 01 — THE SILHOUETTE REVEAL HERO

The first image is not fully visible at load.
A silhouette slowly emerges through darkness, masking, or layered planes as the user scrolls or the page settles.

Goal: immediate intrigue and authority.

WOW MOMENT 02 — THE FACE-TO-FORM TRANSFORMATION SPLIT

A section where the face of the model gradually gives way to the hair silhouette / constructed form through a scroll transition or masked image sequence.

Goal: literally dramatize transformation.

WOW MOMENT 03 — THE PINNED MUTATION STAGE

A pinned section where one central transformation image remains fixed while smaller supporting images orbit, slide, or replace details around it.

Goal: make the portfolio feel spatial and choreographed rather than stacked.

WOW MOMENT 04 — THE CONSTRUCTION DOSSIER

A process section that feels like opening a classified folder: close crops, annotations, structural details, materials, phases.

Goal: prove rigor and add fetish-level craft detail.

WOW MOMENT 05 — THE IDENTITY RUPTURE INTERACTION

A transformation proof module where the visitor actively reveals one identity state from another in a cinematic, restrained way.

Goal: make the transformation feel authored, not simply shown.

WOW MOMENT 06 — THE GALLERY INTERRUPTION QUOTE

Inside the mutation archive, the image rhythm is interrupted by one huge manifesto line or conceptual quote set across black space or translucent material.

Goal: give the site an editorial spine and pacing contrast.

WOW MOMENT 07 — THE FINAL VANISH

The final CTA image or silhouette subtly dissolves / slides / disappears as the user reaches the footer zone, leaving only text and darkness.

Goal: end like a film, not a website.

9. SPATIAL LAYOUT LANGUAGE

The layout system must feel designed around silhouette tension, not generic blocks.

Core spatial rules
Rule 01 — Use asymmetry with discipline

Not every section needs to be broken, but the site should avoid a rigid repeated container pattern.

Rule 02 — Oversize selected images aggressively

One strong image should often dominate a section rather than everything being equal size.

Rule 03 — Allow negative space to carry emotion

Silence is part of the experience.
Do not fill every gap.

Rule 04 — Use narrow and wide rhythm

Alternate between:

compressed narrow columns
expansive full-bleed image moments
floating detail crops
dense and sparse visual beats
Rule 05 — Let images overlap if it adds depth

The project can handle controlled image overlap, especially in hero, manifesto, and archive sections.

10. IMAGE SYSTEM — HOW IMAGES SHOULD BE USED

The image system should feel curated like a fashion monograph.

There are 5 image roles in this world
Role A — Transformation icons

The most powerful hero/editorial images.
Used sparingly and given scale.

Role B — Silhouette studies

Side profiles, back views, obscured faces, line-driven frames.

Role C — Construction evidence

Hands, tools, structures, wig prep, detail work.

Role D — Texture / material closeups

Surface, strands, shine, edge, seam, lacquer, structure.

Role E — Atmospheric still life

Objects, environment, backstage residue, materials.

Rule

Never dump all roles together randomly.
Each section must know which image role is dominant.

13. TYPOGRAPHY BEHAVIOR

Typography must behave like fashion editorial structure, not website UI text.

Typographic roles
Role 01 — World title typography

Large, severe, elegant, iconic.
Used sparingly.

Role 02 — Manifesto / conceptual statements

Can be oversized, fragmented, or broken across lines with intentional rhythm.

Role 03 — Technical captioning

Small, sharp, controlled, almost like museum labels or fashion archive metadata.

Role 04 — CTA typography

Minimal and confident.

Typography should do the following
create tension through scale contrast
punctuate the gallery rhythm
provide breathing space between heavy image acts
feel authored and spatial, not merely readable
Typography should not do the following
become decorative noise
cover too much of the imagery
feel like blog copy
use endless giant serif headlines just because “luxury”

15. SIGNATURE COMPONENT SYSTEM

This site should not rely on ordinary portfolio modules.
It needs a small set of signature components unique to this world.

Component 01 — Transformation Frame

A large image module with layered metadata and optional split-state behavior.

Component 02 — Silhouette Study Block

A narrow, tall portrait block optimized for side-profile / shape reading.

Component 03 — Construction Dossier Card

A process module pairing one technical image with caption / concept / phase label.

Component 04 — Mutation Cluster

A curated group of 2–4 images arranged as one editorial unit, not separate gallery cards.

Component 05 — Identity Split Module

A transformation comparison unit designed specifically for this world.

Component 06 — Atmosphere Strip

A still-life / object / backstage image strip that acts as a pacing reset.

Component 07 — Closing Invocation Panel

The final CTA system, designed like the closing title card of a fashion film.

16. 3D / CANVAS / ADVANCED INTERACTION OPPORTUNITIES

This project can absolutely support elevated interaction, but only if it strengthens the concept.

Good uses of advanced interaction
A. Hero veil reveal

Use layered planes / masks / subtle displacement to make the hero feel like an obscured form coming into view.

B. Floating image depth in the transformation gate

Subtle parallax depth for image fragments, not fake 3D models.

C. Identity split interaction

A cinematic reveal system between two states.

D. Pinned mutation stage

A spatial gallery sequence where one image remains while others transition around it.

Avoid
random 3D object for the sake of “modern portfolio”
excessive WebGL performance tax if the imagery itself is already strong
fancy interaction that weakens the editorial severity

The site should feel expensive, not gimmicky.

17. MOBILE EXPERIENCE RULES

This project cannot collapse into a long stack of centered images and text on mobile.

It must still feel authored.

Mobile rules
preserve oversized image moments where possible
keep silhouette studies tall and dramatic
break text into controlled fragments, not giant paragraphs
archive sequences may need re-authored mobile ordering, not automatic stacking
identity split module must remain elegant and usable on touch
construction dossier can simplify, but should retain its “evidence” feeling
On mobile, prioritize:
hero impact
silhouette readability
archive rhythm
identity split
CTA mood
18. ANTI-TEMPLATE RULES

If any of these happen, the build is drifting into mediocrity.

Instant failure signals
centered hero with heading + button over one image
standard masonry gallery repeated for the whole site
same spacing rhythm for every section
generic fade-up animation on every scroll item
white cards on dark background
service cards with icons
“About / Services / Gallery / Testimonials / Contact” salon structure
stock luxury serif overload
before/after salon slider
ordinary footer with social icons and address block
a build where the imagery is good but the site architecture feels normal
19. BUILD PRIORITY STACK

When this site is actually built, the order of perfection should be:

Priority 01 — Hero transformation gate

If this fails, the whole world collapses.

Priority 02 — Mutation archive image rhythm

The portfolio body must feel authored and cinematic.

Priority 03 — Silhouette chamber

This is one of the main differentiators of the project.

Priority 04 — Identity split module

This is where the transformation thesis becomes explicit.

Priority 05 — Construction dossier

This gives the project credibility and depth.

Priority 06 — Closing invocation

The ending must feel as intentional as the opening.

20. FINAL CREATIVE COMMANDMENT

When building The Avant-Garde Transformer, remember this:

You are not designing a hair stylist website.

You are building a transformation world where hair behaves like:

sculpture
disguise
architecture
mutation
conceptual fashion object
identity engine

The visitor should not leave thinking:
“this stylist does creative/editorial hair.”

They should leave thinking:

this is someone who can re-author a human image from the silhouette outward.

That is the standard.
